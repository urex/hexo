ios webview捕获request及调用本地HTML

By Melface

Published Jul 23 2014

Contents

用HTML来做APP的界面，可以使界面更加的丰富多彩，开发起来也比较方便。
在实现过程中遇到了以下2个问题：

  • HTML如何调用后台IOS代码
    : 首先在接口中申明委托。

    然后在初始化方法中添加委托关系。

    最后实现委托中 shouldStartLoadWithRequest 方法, request提交前会出发此方法，因此判断 request 的值就可以调用不同的 ios 方法了。
  • 如何调用本地HTML
    纯本地应用的话不需要网络，那就没必要去服务器打开HTML页面了，把页面放APP资源文件夹，直接打开就行了。
